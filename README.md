## 📌 Setup & Installation
1. Clone the repository:
   ```sh
   git clone https://github.com/TPashaxrd/restaurant-system-qrcode.git
   ```
2. Navigate to the project folder:
   ```sh
   cd restaurant-qr-system
   ```
3. Install dependencies:
   ```sh
   npm install
   ```
4. Run the project:
   ```sh
   npm run dev
   ```
